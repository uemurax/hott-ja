4.1 始代数[0031]

  • 定義 4.1.1 ([0035]). \(i\)を階数とする。型\(\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i) :\mathcal {U}(\mathord {\text {\textsf {suc}}}(i))\)を次のレコード型と定義する。

    • • \(\mathord {\text {\textsf {Carrier}}}:\mathcal {U}(i)\)

    • • \(\mathord {\text {\textsf {zero}}}:\mathord {\text {\textsf {Carrier}}}\)

    • • \(\mathord {\text {\textsf {suc}}}:\mathord {\text {\textsf {Carrier}}}\to \mathord {\text {\textsf {Carrier}}}\)

    \(\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)の要素を (階数\(i\)の) \(\protect \mathbb {N}\)代数(\(\mathbb {N}\)-algebra) と呼ぶ。

  • 定義 4.1.2 ([0036]). \(i\)を階数、\(A, B :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)を\(\mathbb {N}\)代数とする。型\(\mathord {\text {\textsf {Hom}}}(A, B) :\mathcal {U}(i)\) を次 のレコード型と定義する。

    • • \(\mathord {\text {\textsf {carrier}}}:A .\mathord {\text {\textsf {Carrier}}}\to B .\mathord {\text {\textsf {Carrier}}}\)

    • • \(\mathord {\text {\textsf {zero}}}:\mathord {\text {\textsf {carrier}}}(A .\mathord {\text {\textsf {zero}}}) =B .\mathord {\text {\textsf {zero}}}\)

    • • \(\mathord {\text {\textsf {suc}}}:\prod _{x :A .\mathord {\text {\textsf {Carrier}}}}\mathord {\text {\textsf {carrier}}}((A .\mathord {\text {\textsf {suc}}})(x)) =(B .\mathord {\text {\textsf {suc}}})(\mathord {\text {\textsf {carrier}}}(x))\)

    \(\mathord {\text {\textsf {Hom}}}(A, B)\)の要素を\(A\)から\(B\)への\(\protect \mathbb {N}\)準同型写像(\(\mathbb {N}\)-homomorphism) と呼ぶ。

  • 演習 4.1.3 ([0037]). \(i\)を階数とする。

    • 1. \(\mathbb {N}\)代数\(A :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)に対して、恒等射\(\mathord {\text {\textsf {id}}}(A) :\mathord {\text {\textsf {Hom}}}(A, A)\)を構成せよ。

    • 2. \(\mathbb {N}\)代数\(A, B, C :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)と\(\mathbb {N}\)準同型写像\(f :\mathord {\text {\textsf {Hom}}}(A, B)\)と\(g :\mathord {\text {\textsf {Hom}}}(B, C)\)に対して、合成\(g \circ f :\mathord {\text {\textsf {Hom}}}(A, C)\)を構成せよ。

  • 定義 4.1.4 ([0038]). \(i\)を階数、\(A :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)を\(\mathbb {N}\)代数とする。型\(\mathord {\text {\textsf {IsInit}}}(A) :\mathcal {U}(\mathord {\text {\textsf {suc}}}(i))\)を\(\prod _{X :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)}\mathord {\text {\textsf {IsContr}}}(\mathord {\text {\textsf {Hom}}}(A, X))\)と定義する。\(\mathord {\text {\textsf {IsInit}}}(A)\)の要素がある時、\(A\)は始\(\protect \mathbb {N}\)代数(initial \(\mathbb {N}\)-algebra) であると言う。

  • 定義 4.1.5 ([0039]). \(i\)を階数、\(A :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)を\(\mathbb {N}\)代数とする。型\(\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A} :\mathcal {U}(\mathord {\text {\textsf {suc}}}(i))\)を次のレコード型と定義する。

    • • \(\mathord {\text {\textsf {Carrier}}}:A .\mathord {\text {\textsf {Carrier}}}\to \mathcal {U}(i)\)

    • • \(\mathord {\text {\textsf {zero}}}:\mathord {\text {\textsf {Carrier}}}(A .\mathord {\text {\textsf {zero}}})\)

    • • \(\mathord {\text {\textsf {suc}}}:\prod _{\lbrack x :A .\mathord {\text {\textsf {Carrier}}} \rbrack }\mathord {\text {\textsf {Carrier}}}(x) \to \mathord {\text {\textsf {Carrier}}}((A .\mathord {\text {\textsf {suc}}})(x))\)

    \(\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}\)の要素を\(A\)上の\(\protect \mathbb {N}\)代数(\(\mathbb {N}\)-algebra over \(A\)) と呼ぶ。

  • 定義 4.1.6 ([003A]). \(i\)を階数、\(A :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)を\(\mathbb {N}\)代数、\(B :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}\)を\(A\)上の\(\mathbb {N}\)代数とする。型\(\mathord {\text {\textsf {Sect}}}(B) :\mathcal {U}(i)\)を次のレコード型と定義する。

    • • \(\mathord {\text {\textsf {carrier}}}:\prod _{x :A .\mathord {\text {\textsf {Carrier}}}}(B .\mathord {\text {\textsf {Carrier}}})(x)\)

    • • \(\mathord {\text {\textsf {zero}}}:\mathord {\text {\textsf {carrier}}}(A .\mathord {\text {\textsf {zero}}}) =B .\mathord {\text {\textsf {zero}}}\)

    • • \(\mathord {\text {\textsf {suc}}}:\prod _{x :A .\mathord {\text {\textsf {Carrier}}}}\mathord {\text {\textsf {carrier}}}((A .\mathord {\text {\textsf {suc}}})(x)) =(B .\mathord {\text {\textsf {suc}}}) (\mathord {\text {\textsf {carrier}}}(x))\)

    \(\mathord {\text {\textsf {Sect}}}(B)\)の要素を\(B\)の切断(section) と呼ぶ。

  • 例 4.1.7 ([003B]). \(i\)を階数、\(A, B :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)を\(\mathbb {N}\)代数とする。\(A\)上の\(\mathbb {N}\)代数 \(A^{{\ast }}B :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}\)を次のように定義する。

  • 例 4.1.8 ([003G]). \(i\)を階数、\(A :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)を\(\mathbb {N}\)代数、\(B :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}\)を\(A\)上の\(\mathbb {N}\)代数、\(f, g :\mathord {\text {\textsf {Sect}}}(B)\)を切断とす る。\(A\)上の\(\mathbb {N}\)代数\(\mathord {\text {\textsf {Id}}}(f, g) :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}\)を次のように定義する。

    • • \(\mathord {\text {\textsf {suc}}}\equiv \lambda x y . (g .\mathord {\text {\textsf {suc}}})(x)^{-1} \circ (B .\mathord {\text {\textsf {suc}}})(y) \circ (f .\mathord {\text {\textsf {suc}}})(x)\)

  • 演習 4.1.9 ([003H]). \(i\)を階数、\(A :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}(i)\)を\(\mathbb {N}\)代数、\(B :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}\)を\(A\)上の\(\mathbb {N}\)代数、\(f :\mathord {\text {\textsf {Sect}}}(B)\)を切断とする。

  • 証明. 2から1は例 4.1.7から従う。

    1から2を示す。\(B :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}\)とす る。仮定より、\(\mathord {\text {\textsf {Hom}}}(A, \mathord {\text {\textsf {Total}}}(B))\)と\(\mathord {\text {\textsf {Hom}}}(A, A)\)は可縮であ る。よって、演習 4.1.12より、関数\(\lambda f . \mathord {\text {\textsf {pr}}}(B) \circ f :\mathord {\text {\textsf {Hom}}}(A, \mathord {\text {\textsf {Total}}}(B)) \to \mathord {\text {\textsf {Hom}}}(A, A)\)は同値である。すると、演習 4.1.11と命題 3.1.6より\(\mathord {\text {\textsf {Sect}}}(B)\)は可縮である。

    関数外延性を仮定し、\(h :\prod _{X :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}}\mathord {\text {\textsf {Sect}}}(X)\)を仮定する。\(B :\mathbb {N}\mathord {\text {\textsf {-}}}\mathord {\text {\textsf {Alg}}}_{/A}\)に対して、\(\mathord {\text {\textsf {Sect}}}(B)\)は可縮であること を示す。\(h(B) :\mathord {\text {\textsf {Sect}}}(B)\)を得る。任意の\(f :\mathord {\text {\textsf {Sect}}}(B)\)に対して、同一視\(p(f) :h(B) =f\)を構成すればよい。演習 4.1.9より、\(p'(f) :\mathord {\text {\textsf {Sect}}}(\mathord {\text {\textsf {Id}}}(h(B), f))\)を構成すればよいが、\(p'(f) \equiv h(\mathord {\text {\textsf {Id}}}(h(B), f))\)とすればよい。□
